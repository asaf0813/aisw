JDK : 자바를 해석하기위한 도구 > 오라클홈페이지

에디터 : 문서를 작성하기 위한 도구 > 네이버 검색

띄어쓰기하면 자동으로 {} 나옴
대문자쓰면 이어쓰기되어있는거임

저장할때 제목없음으로 저장하면 큰일남(대소문자구문도해야함, Class의 이름과 동일해야함)

JAVAC : 컴파일 시켜주는 것 ( class를 만들어주는 것 )  ctrl 1

JAVA : 자바를 실행시켜주기 위한 것임 실행하려면 ( class를 실행시켜줌 ) ctrl 2
(컴파일을 하지 않으면 실행시켜도 마지막 컴파일 된게 불러와짐)

파일 > 새파일 > JAVA
구문이 생기면 class 앞에 public을 붙임

해석하려면 class파일로 변경되어야 컴퓨터가 해석할 수 있음
